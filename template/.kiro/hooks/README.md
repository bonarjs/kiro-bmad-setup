# BMAD Automation Hooks

This directory contains Kiro hooks that automate BMAD processes and workflows.

## Structure

Hook files follow the naming pattern: `bmad-{functionality}.hook`

## Available Hooks

- `bmad-test-sync.hook` - Automatic test synchronization
- `bmad-doc-sync.hook` - Documentation updates
- `bmad-code-standards.hook` - Code standards enforcement
- `bmad-story-validation.hook` - Story format validation

## Usage

Hooks are automatically triggered by Kiro based on file changes and development events.
